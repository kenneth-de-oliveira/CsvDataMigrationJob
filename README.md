# Csv Migration Batch - POC 

This project is a proposal for a JOB for automatic migration of lines from a CSV file to a DATABASE using the following technologies:

* _Java 21_
* _Maven 3.8.3_
* _Mysql 8.0.26_
* Spring Boot 
* Spring Batch
* Lombok
